# 屏幕保护时钟 🕐

一个单 HTML 文件的全屏时钟页面，支持屏幕保护模式和个性化设置。在浏览器中直接打开即可使用，无需服务器、无需构建。

## 功能

- ⏰ **全屏时钟** — 时间占据视口约 80%，每秒更新，支持 12/24 小时制
- 🎨 **3 种预设主题** — 暗夜模式 / 极简白 / 复古琥珀，一键切换
- ⚙️ **个性化设置** — 字体大小、背景颜色、文字颜色、字体族、时区
- 🖥️ **屏幕保护模式** — 开启后时钟在屏幕上缓慢移动并反弹，防止烧屏
- 💾 **自动保存** — 所有设置持久化到浏览器 localStorage

## 使用

1. 下载 `index.html`
2. 双击用浏览器打开
3. 点击屏幕任意位置 → 右上角出现 ⚙ 设置图标 → 点击进入设置

或使用 Caddy 本地服务：

```bash
caddy run
# 访问 http://localhost:8080
```

## 预览

打开页面后默认显示暗夜模式黑色背景的大时钟。点击空白处呼出设置面板，可切换主题为极简白或复古琥珀。

## 技术

- 纯原生 HTML + CSS + JavaScript
- 零外部依赖
- 设置存储在 localStorage
- 时区转换使用 `Intl.DateTimeFormat` API
